# Big Number Matrix Operator  

## Explain  
   This is Lab4 of course OOP

## How to build  
1. build main  
   `make`  
2. run main  
   `make run`  
3. build cmder tool  
   `make cmder`  
4. clean  
   `make clean`  

## How to use  
1. for main  
   Input file:  
   matrix.in  
   expression.in  

   Output file:  
   result.out  

   you can change default file in FileUtil.h  

2. for cmder  
   use  
   `cd bin`  
   `./cmder`  
   to run  

   type "help" get help message
